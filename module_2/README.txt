Основная цель проекта - отследить влияние условий жизни учащихся на их успеваемость по математике.
Данные содержат в себе информацию об условиях жизни, родителях, условиях обучения учащихся.
Этапы работы над проектом:
1. Определение функций, которые будем использовать 
2. Первичный осмотр данных
3. Анализ числовых и номинативных признаков
4. Преобразование данных
5. Построение корреляции числовых признаков
6. Проверка наличия статистической разницы при помощи теста Стьюдента
7. Определение датасета для построения модели
8. Итоговые выводы
